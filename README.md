	КРАТКОЕ ОПИСАНИЕ ЛАБОРАТОРНОЙ РАБОТЫ: СТРУКТУРЫ. БАЗА СТУДЕНТОВ

**Цели работы:** изучение и организация структур; получение практических навыков работы со структурами; определение преимуществ и недостатков использования структур.

**Функционал:**

➕ Создание записи (ФИО, пол, группа, номер в списке, 8 оценок)<br>
✏️ Редактирование записи<br>
📄 Вывод всех студентов<br>
🔍 Фильтр по группе<br>
🏆 Топ-3 по среднему баллу<br>
♀️♂️ Статистика по полу<br>
🎓 Фильтрация по успеваемости<br>
🔢 Поиск по номеру в списке<br>
💾 Автосохранение в текстовый файл

🗂️ **Структура проекта**

main.cpp — основной код<br>
database.txt — файл с данными студентов<br>
competition.txt — файл с командами (ИДЗ)<br>

**Каждая запись содержит:** <br>
ID, <br>
ФИО, <br>
пол (M/F), <br>
группу, <br>
8 оценок (2-5), <br>
дату создания<br>
