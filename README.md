	КРАТКОЕ ОПИСАНИЕ ЛАБОРАТОРНОЙ РАБОТЫ: СТРУКТУРЫ. БАЗА СТУДЕНТОВ

**Цели работы:** изучение и организация структур; получение практических навыков работы со структурами; определение преимуществ и недостатков использования структур.

**Функционал:**

➕ Создание записи (ФИО, пол, группа, номер в списке, 8 оценок)
✏️ Редактирование записи
📄 Вывод всех студентов
🔍 Фильтр по группе
🏆 Топ-3 по среднему баллу
♀️♂️ Статистика по полу
🎓 Фильтрация по успеваемости (неуспевающие/хорошисты/отличники)
🔢 Поиск по номеру в списке
💾 Автосохранение в текстовый файл

🗂️ **Структура проекта**
main.cpp — основной код
database.txt — файл с данными студентов
competition.txt — файл с командами (ИДЗ)

Каждая запись содержит: 
ID, 
ФИО, 
пол (M/F), 
группу, 
8 оценок (2-5), 
дату создания
